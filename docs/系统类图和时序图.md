## 二、登录时序图

```mermaid
sequenceDiagram
    participant User as 用户
    participant AuthController as AuthController
    participant AuthenticationManager as AuthenticationManager
    participant UserService as UserService
    participant JwtTokenProvider as JwtTokenProvider
    participant UserMapper as UserMapper
    participant MySQL as MySQL
    
    User->>AuthController: POST /auth/login (username, password)
    AuthController->>AuthenticationManager: authenticate(UsernamePasswordAuthenticationToken)
    AuthenticationManager->>UserService: loadUserByUsername(username)
    UserService->>UserMapper: selectOne(wrapper)
    UserMapper->>MySQL: SELECT * FROM users WHERE username = ?
    MySQL-->>UserMapper: 返回用户信息
    UserMapper-->>UserService: 返回User对象
    UserService-->>AuthenticationManager: 返回UserDetails
    
    alt 认证成功
        AuthenticationManager-->>AuthController: 返回Authentication对象
        AuthController->>SecurityContextHolder: setAuthentication(authentication)
        AuthController->>JwtTokenProvider: generateToken(userDetails)
        JwtTokenProvider-->>AuthController: 返回JWT Token
        AuthController->>UserService: findByUsername(username)
        UserService->>UserMapper: selectOne(wrapper)
        UserMapper->>MySQL: SELECT * FROM users WHERE username = ?
        MySQL-->>UserMapper: 返回用户信息
        UserMapper-->>UserService: 返回User对象
        UserService-->>AuthController: 返回User对象
        AuthController-->>User: 200 OK { "token": "...", "user": {...} }
    else 认证失败
        AuthenticationManager-->>AuthController: 抛出BadCredentialsException
        AuthController-->>User: 401 Unauthorized { "error": "用户名或密码错误" }
    end
```

## 三、选题提交时序图

```mermaid
sequenceDiagram
    participant Student as 学生
    participant TopicSelectionController as TopicSelectionController
    participant TopicSelectionService as TopicSelectionService
    participant UserService as UserService
    participant TopicSelectionMapper as TopicSelectionMapper
    participant MySQL as MySQL
    
    Student->>TopicSelectionController: POST /topic/student/submit (topicSelection)
    TopicSelectionController->>TopicSelectionService: submitTopic(topicSelection)
    TopicSelectionService->>UserService: findById(studentId)
    UserService->>UserMapper: selectById(studentId)
    UserMapper->>MySQL: SELECT * FROM users WHERE user_id = ?
    MySQL-->>UserMapper: 返回用户信息
    UserMapper-->>UserService: 返回User对象
    UserService-->>TopicSelectionService: 返回User对象
    
    alt 学生已绑定导师
        TopicSelectionService->>TopicSelectionMapper: insert(topicSelection)
        TopicSelectionMapper->>MySQL: INSERT INTO topic_selections (...) VALUES (...)
        MySQL-->>TopicSelectionMapper: 插入成功
        TopicSelectionMapper-->>TopicSelectionService: 返回插入结果
        TopicSelectionService-->>TopicSelectionController: 选题提交成功
        TopicSelectionController-->>Student: 200 OK "自拟题目提交成功，等待老师审核"
    else 学生未绑定导师
        TopicSelectionService-->>TopicSelectionController: 抛出RuntimeException
        TopicSelectionController-->>Student: 400 Bad Request "请先绑定导师"
    end
```

## 四、论文提交时序图

```mermaid
sequenceDiagram
    participant Student as 学生
    participant ThesisProcessController as ThesisProcessController
    participant PaperService as PaperService
    participant FileService as FileService
    participant PaperMapper as PaperMapper
    participant FileMapper as FileMapper
    participant MySQL as MySQL
    
    Student->>ThesisProcessController: POST /thesis/student/submit-paper (paper)
    ThesisProcessController->>PaperService: submitPaper(paper)
    PaperService->>FileService: getFileById(fileId)
    FileService->>FileMapper: selectById(fileId)
    FileMapper->>MySQL: SELECT * FROM files WHERE file_id = ?
    MySQL-->>FileMapper: 返回文件信息
    FileMapper-->>FileService: 返回File对象
    FileService-->>PaperService: 返回File对象
    
    alt 文件存在
        PaperService->>PaperMapper: insert(paper)
        PaperMapper->>MySQL: INSERT INTO papers (...) VALUES (...)
        MySQL-->>PaperMapper: 插入成功
        PaperMapper-->>PaperService: 返回插入结果
        PaperService-->>ThesisProcessController: 论文提交成功
        ThesisProcessController-->>Student: 200 OK { paper对象 }
    else 文件不存在
        PaperService-->>ThesisProcessController: 抛出RuntimeException
        ThesisProcessController-->>Student: 400 Bad Request "文件不存在"
    end
```

## 五、论文审核与评分时序图

```mermaid
sequenceDiagram
    participant Teacher as 教师
    participant ThesisProcessController as ThesisProcessController
    participant PaperService as PaperService
    participant PaperMapper as PaperMapper
    participant MySQL as MySQL
    
    Teacher->>ThesisProcessController: POST /thesis/teacher/grade-paper (paperId, score, comment)
    ThesisProcessController->>PaperService: gradePaper(paperId, score, comment)
    PaperService->>PaperMapper: selectById(paperId)
    PaperMapper->>MySQL: SELECT * FROM papers WHERE paper_id = ?
    MySQL-->>PaperMapper: 返回论文信息
    PaperMapper-->>PaperService: 返回Paper对象
    
    alt 论文存在
        PaperService->>PaperMapper: updateById(paper)
        PaperMapper->>MySQL: UPDATE papers SET score = ?, review_opinion = ?, review_status = 'APPROVED', review_time = NOW() WHERE paper_id = ?
        MySQL-->>PaperMapper: 更新成功
        PaperMapper-->>PaperService: 返回更新结果
        PaperService-->>ThesisProcessController: 论文评分成功
        ThesisProcessController-->>Teacher: 200 OK "Paper graded successfully"
    else 论文不存在
        PaperService-->>ThesisProcessController: 抛出RuntimeException
        ThesisProcessController-->>Teacher: 404 Not Found "论文不存在"
    end
```
